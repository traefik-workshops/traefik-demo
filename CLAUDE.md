# Agent guide

The canonical guide for agents in this repo is [`AGENTS.md`](./AGENTS.md). Read it before touching anything.

`AGENTS.md` is the cross-tool convention (Claude Code, Codex, Cursor, Gemini CLI all read it). This file is preserved so Claude Code still surfaces a pointer if it auto-loads `CLAUDE.md`; all canonical content lives in `AGENTS.md` (and per-section `AGENTS.md` files).
