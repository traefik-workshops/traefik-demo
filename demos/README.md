# demos/

Compositions of the library in this repo, in two tiers (see [`AGENTS.md`](./AGENTS.md) for the full convention):

- **Archetypes** ‚Äî minimal, sanitized Terraform compositions that show how the modules and charts wire together. Not meant to run end-to-end; they are the shapes the `sa-assistant` skill pattern-matches when building a PoC, and a quick sanity-check that a module refactor still composes.
- **Runnable demos** ‚Äî self-contained, k3d-based, white-labeled demos you can bring up locally for free. Each ships a `Makefile` + `scenarios.sh`.

Module sources are **relative** (`../../terraform/...`) so every demo tracks the library in this checkout and `terraform validate` runs offline. (Standalone PoCs generated by the `build-poc` skill pin `?ref=<version>` instead.)

## Conventions

Every archetype ships:

- `main.tf` ‚Äî the module composition (this is the file an agent should pattern-match against)
- `variables.tf` ‚Äî inputs the demo expects
- `terraform.tfvars.example` ‚Äî placeholder values. **Never commit real values.**
- `outputs.tf` ‚Äî the canonical Traefik IP + dashboard URL + any module pass-throughs
- `README.md` ‚Äî what the demo proves, install steps, how to extend

Runnable demos additionally ship `versions.tf`, a `Makefile` (`up`/`down`/`scenarios`), and `scenarios.sh` (curl assertions vs an expected table). See [`AGENTS.md`](./AGENTS.md).

## What these are NOT

- Production-grade. They use the smallest cluster size, in-cluster databases, and demo credentials.
- A complete tutorial. They expect the reader to know Terraform + Helm basics.
- Tied to the real demos. If a real demo diverges, that demo is the source of truth ‚Äî these are just the canonical shapes.

## Module sources

Demos use **relative** sources (`../../terraform/<section>/<path>`) ‚Äî see [`AGENTS.md`](./AGENTS.md). They intentionally do *not* pin a `?ref=<tag>`: a demo tracks the library in this checkout, so `terraform validate` runs offline and always against the current modules. The `build-poc` skill pins a version when it emits a standalone PoC for a prospect.
